Aspects Techniques de CodeQuest

1. Structure du Projet

Le projet est organisé en plusieurs fichiers HTML pour chaque chapitre, avec un fichier CSS centralisé pour les styles, et des fichiers JavaScript pour la logique.

2. Technologies Utilisées

  • HTML/CSS : Pour la structure et le design des pages.
  • JavaScript : Pour l'interactivité, y compris la vérification des défis et la gestion des comptes.
  • Firebase : Pour la gestion des comptes utilisateurs, l'authentification et la sauvegarde de l'avancement.
  • Cookies : Pour stocker les informations de session utilisateur.

3. Gestion des Comptes Utilisateurs

Nous utilisons Firebase pour l'authentification sécurisée, intégrée via l'API Firebase. Le code suivant initialise Firebase :

// Configuration de Firebase
const firebaseConfig = {
  apiKey: "Votre-API-Key",
  authDomain: "code-quest.firebaseapp.com",
  projectId: "code-quest",
  storageBucket: "code-quest.appspot.com",
  messagingSenderId: "123456789",
  appId: "1:123456789:web:abcdefg",
};

// Initialisation de Firebase
const app = initializeApp(firebaseConfig);

4. Stockage de l'Avancement

L'avancement des utilisateurs est stocké dans Firebase Realtime Database. Les cookies sont utilisés pour suivre la session utilisateur, permettant une expérience personnalisée lors de la reconnexion.

5. Défis et Validation

Les défis sont validés via JavaScript. Voici un exemple de vérification simple pour les défis :

submitBtn.addEventListener("click", function() {
    const code = editor.value.trim();
    if (code === "x = 20") {
        result.textContent = "Bravo !";
        setTimeout(() => window.location.href = "next_chapter.html", 2000);
    } else {
        result.textContent = "Essayez encore !";
    }
});

6. Gestion des Styles

Le design du projet est inspiré des anciens terminaux, avec des couleurs sombres et une police monospaced. Le CSS est écrit pour un style minimaliste et rétro.
