Chapitre 1 : Variables et Opérations

Introduction aux Variables

Une variable est un espace de mémoire utilisé pour stocker une valeur qui peut être utilisée et modifiée au cours du programme.

x = 10  # x est une variable contenant la valeur 10

Les variables en Python ne nécessitent pas de déclaration explicite de type. Vous pouvez stocker différents types de données dans des variables, tels que des nombres, des chaînes de caractères, des listes, etc.

Types de Données

Voici quelques types de données courants en Python :

  • Entiers : Représentent des nombres sans décimales. Exemple : x = 5
  • Flottants : Représentent des nombres avec des décimales. Exemple : y = 3.14
  • Chaînes de Caractères : Représentent des textes. Exemple : nom = "Alice"
  • Booléens : Représentent des valeurs de vérité. Exemple : est_vrai = True

# Exemple de variables
nombre_entier = 10
nombre_flottant = 3.14
texte = "Bonjour"
booleen = True

Opérations de Base

Les variables peuvent être utilisées pour effectuer des opérations de base :

  • Addition : somme = a + b
  • Soustraction : difference = a - b
  • Multiplication : produit = a * b
  • Division : quotient = a / b

a = 5
b = 3
somme = a + b  # Résultat : 8
difference = a - b  # Résultat : 2
produit = a * b  # Résultat : 15
quotient = a / b  # Résultat : 1.6667
