Chapitre 2 : Boucles, Conditions et Fonctions

Introduction aux Boucles

Les boucles permettent d'exécuter un bloc de code plusieurs fois. Bien que nous ne couvrions pas encore toutes les boucles, nous allons vous montrer comment utiliser une boucle for en utilisant les concepts appris dans le Chapitre 1.

Boucle for simple

for i in range(5):
    print(i)  # Affiche les nombres de 0 à 4

Explication : Cette boucle affiche les nombres de 0 à 4. Le mot-clé range(5) génère une séquence de nombres de 0 à 4.

Introduction aux Conditions

Les conditions permettent de diriger le flux du programme en fonction de certaines valeurs. Voici comment utiliser une condition if avec des variables.

x = 10
if x > 5:
    print("x est supérieur à 5")
else:
    print("x est inférieur ou égal à 5")

Explication : Cette condition vérifie si x est supérieur à 5 et affiche un message en conséquence.

Fonctions Simples

Nous allons maintenant créer une fonction simple en utilisant les concepts de variables et opérations. Vous allez écrire une fonction pour additionner deux nombres.

def additionner(a, b):
    return a + b

Explication : La fonction additionner prend deux paramètres, les additionne et retourne le résultat.
