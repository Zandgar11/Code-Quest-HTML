Aspects Techniques de CodeQuest

1. Structure du Projet

Le projet est structuré avec des fichiers HTML pour chaque chapitre, un fichier CSS centralisé pour les styles, et des fichiers JavaScript pour les fonctionnalités.

2. Technologies Utilisées

  • HTML/CSS : Pour la structure et le design des pages.
  • JavaScript : Pour l'interactivité, incluant la gestion des défis et des comptes utilisateurs.
  • Cookies : Pour la gestion des sessions utilisateur et la sauvegarde de l'avancement.

3. Gestion des Comptes Utilisateurs

Nous utilisons les cookies pour stocker les informations des utilisateurs. Voici un exemple de code pour la gestion des cookies :

function setCookie(name, value, days) {
    let expires = "";
    if (days) {
        const date = new Date();
        date.setTime(date.getTime() + (days*24*60*60*1000));
        expires = "; expires=" + date.toUTCString();
    }
    document.cookie = name + "=" + (value || "")  + expires + "; path=/";
}

function getCookie(name) {
    const nameEQ = name + "=";
    const ca = document.cookie.split(';');
    for(let i=0;i < ca.length;i++) {
        let c = ca[i];
        while (c.charAt(0)==' ') c = c.substring(1,c.length);
        if (c.indexOf(nameEQ) == 0) return c.substring(nameEQ.length,c.length);
    }
    return null;
}

4. Stockage de l'Avancement

L'avancement des utilisateurs est enregistré dans les cookies pour persister les données entre les sessions.

5. Défis et Validation

Les défis sont validés par des scripts JavaScript côté client. Voici un exemple de vérification simple pour un défi :

submitBtn.addEventListener("click", function() {
    const code = editor.value.trim();

    }
});

6. Gestion des Styles

Le design du projet est inspiré des anciens terminaux, avec des couleurs sombres et une police monospaced. Le CSS est écrit pour un style minimaliste et rétro.
