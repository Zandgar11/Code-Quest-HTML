Chapitre 1 : Variables et Opérations

Introduction aux Variables

Une variable est un espace de mémoire utilisé pour stocker une valeur qui peut être utilisée et modifiée au cours du programme.

x = 10  # x est une variable contenant la valeur 10

Types de Données en Python

Python prend en charge plusieurs types de données de base :

  • int : Nombre entier. Ex: x = 5
  • float : Nombre à virgule flottante. Ex: y = 3.14
  • str : Chaîne de caractères. Ex: nom = "Alice"
  • bool : Valeur booléenne (Vrai ou Faux). Ex: est_vrai = True

Vous pouvez vérifier le type d'une variable en utilisant la fonction type() :

type(x)  # Retourne

Opérations de Base avec les Variables

En Python, vous pouvez effectuer des opérations arithmétiques de base comme :

  • + : Addition
  • - : Soustraction
  • * : Multiplication
  • / : Division
  • % : Modulo (reste de la division)
a = 10
b = 3
somme = a + b  # Résultat : 13
produit = a * b  # Résultat : 30
reste = a % b  # Résultat : 1
